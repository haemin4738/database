# 데이터베이스 항공 사이트 제작 ✈️

## 📅 프로젝트 기간
- 2023년 05월

## 📌 프로젝트 개요
대한항공 사이트를 레퍼런스로 삼아 항공권 예약 시스템을 제작한 프로젝트입니다.  
HTML, CSS, JavaScript, PHP 기반으로 백엔드와 프론트엔드를 연동하여 사용자들이 항공권을 조회하고 예약할 수 있도록 구현했습니다.

## 👨‍👩‍👧‍👦 역할
- 팀장

## 🛠 기술 스택
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  

## 📝 주요 업무
- 백엔드 API 설계 및 구현
- 프론트엔드 개발 지원
- 팀원 간 커뮤니케이션 및 문제 해결
